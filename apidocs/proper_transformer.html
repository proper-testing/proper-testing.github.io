Description

PropEr's main parse transform. It is automatically applied to modules when including the main PropEr header, unless PROPER_NO_TRANS is defined. Applying this transform has the following effects:
  • All 0-arity functions whose name begins with prop_ are automatically exported.
  • Type declarations in ?FORALLs that correspond to native types are properly substituted (with some limitations, see the proper_typeserver module for details).
